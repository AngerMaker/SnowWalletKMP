# SnowWallet ❄️💳

O **SnowWallet** é um aplicativo de controle financeiro pessoal multiplataforma. Desenvolvido com o intuito de facilitar a gestão de gastos, receitas, cartões de crédito e empréstimos, o app foi construído utilizando tecnologias modernas do ecossistema Android e Kotlin.

Este projeto faz parte do meu portfólio pessoal como desenvolvedor e explora o compartilhamento de código entre plataformas (Android e Desktop) usando Kotlin Multiplatform (KMP) e Compose Multiplatform.

## ✨ Funcionalidades

* **Gestão de Transações:** Adicione, edite e acompanhe suas receitas e despesas.
* **Controle de Contas e Categorias:** Organize seu dinheiro dividindo-o em diferentes contas e categorize seus gastos.
* **Gestão de Cartões de Crédito e Faturas:** Acompanhe o limite e as faturas dos seus cartões.
* **Empréstimos:** Controle de valores emprestados ou tomados como empréstimo.
* **Transações Recorrentes:** Agendamento e controle de gastos/ganhos fixos.
* **Notificações:** Alertas locais para lembrar de pagamentos e faturas.
* **Backup em Nuvem:** Integração com o Google Drive para salvar e restaurar seus dados.
* **Widgets (Android):** Atalhos rápidos na tela inicial para adicionar despesas e receitas, e visualizar o saldo.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** [Kotlin](https://kotlinlang.org/)
* **UI:** [Compose Multiplatform](https://www.jetbrains.com/lp/compose-multiplatform/)
* **Arquitetura:** MVVM (Model-View-ViewModel)
* **Injeção de Dependência:** [Koin](https://insert-koin.io/)
* **Banco de Dados Local:** Room / SQLite
* **Navegação:** Voyager (ou a biblioteca de navegação KMP que você utilizou)
* **Coroutines & Flow:** Para programação assíncrona e reativa.

## 🚀 Como Executar o Projeto

### Pré-requisitos
* [Android Studio](https://developer.android.com/studio) (versão mais recente recomendada) ou IntelliJ IDEA.
* JDK 17 ou superior.

### Passos
1. Clone este repositório:
   ```bash
   git clone [https://github.com/SEU_USUARIO/snow-wallet.git](https://github.com/SEU_USUARIO/snow-wallet.git)

2. Abra o projeto no Android Studio ou IntelliJ IDEA.

3. Aguarde o Gradle sincronizar todas as dependências.

4. Para rodar no Android: Selecione a configuração composeApp ou androidApp e um emulador/dispositivo físico e clique em Run.

5. Para rodar no Desktop: Execute a task do Gradle desktopRun ou selecione a configuração correspondente na IDE.
